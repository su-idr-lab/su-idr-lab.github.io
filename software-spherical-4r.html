Teaching App

This interactive spherical 4R linkage application is intended for teaching and exploration of spherical mechanism geometry, motion, and design relationships. It provides a visual environment for understanding how a spherical 4R linkage moves and how its parameters shape the resulting kinematics.

The linked code repository and AI Studio deployment suggest a broader interactive mechanism-design workflow in which geometric intuition, quick visualization, and user-driven parameter changes are central to the learning experience.
