Legacy Software

MagicDNA is a DNA-origami design tool for multi-component assemblies, packaged as a MATLAB app and also distributed for use with MATLAB Runtime. According to the project README, it provides a graphical interface guided by computation for the design of DNA origami structures and supports tutorial materials, workshop slides, and a dedicated tutorial video channel.

MagicDNA is particularly significant because it bridges computer-aided engineering and DNA nanostructure design, turning complex nanoscale assemblies into a more accessible design workflow.
