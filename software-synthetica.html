Legacy Software

Synthetica is a spatial linkage synthesis tool referenced in both the legacy software ecosystem and the machine-design automation work of the lab. It is associated with an extensible Java applet for spatial linkage synthesis and represents an earlier generation of interactive computational support for mechanism design.

Its importance lies in supporting kinematic synthesis workflows for spatial linkages, which aligns closely with the lab’s long-running interests in computational kinematics and design automation.
