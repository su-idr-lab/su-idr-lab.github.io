# IDR Lab Website

Starter static website for the `su-idr-lab` GitHub Pages organization site.

## Expected publish target

- Repository name: `su-idr-lab.github.io`
- Site URL: `https://su-idr-lab.github.io/`

## Files

- `index.html`
- `people.html`
- `research.html`
- `publications.html`
- `software.html`
- `join.html`
- `assets/css/style.css`

## Logo

The site expects the lab logo file at:

- `logos/IDRLab_logo2_small_v2.png`

If that file is not present yet, create the `logos/` folder and place the image there before publishing.

## Deploy

1. Create the repository `su-idr-lab.github.io` under the `su-idr-lab` organization.
2. Copy these files into the repository root.
3. Commit and push to `main`.
4. In GitHub repository settings, enable Pages from the `main` branch and `/root`.
